PSIrc documentation
===================

This is a documentation for the PSIrc project.
PSIrc is a simple IRC Server that supports:

* Users
* Operators
* Channel operators
* IRC operators
* Private messaging
* Channel messaging
* Kicking out of channels

The server follows RFC 1459 and mostly works with IRC clients that support it as well
